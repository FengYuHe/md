服务端接口文档
========
## 接口地址
公共接口访问地址是：http://localhost/rest

## 接口列表
### User
* [`POST /user/login`](#login)(用户)


### Devices
* [`GET /node/:nid/activate`](#activate)(设备)
* [`POST /node/:nid/data`](#postdata)(设备)
* [`GET /node/:nid/commands`](#commands)(设备)
* [`POST /node`](#claim)(用户)
* [`PUT /node/:nid/data`](#putdata)(用户)
* [`GET /node`](#getAllDevice)(用户)
* [`GET /node/:nid`](#getDevice)(用户)
* [`DELETE /node/:nid`](#deleteDevice)(用户)

## User

<a name="login" />
### 获取access_token
**[POST] /user/login**
#### 参数
* `username`  用户名
* `password`  密码

#### Example
```sh
curl -X 'POST' \
-H 'Content-Type: application/json' \
-d '{"username":"123","password":"123"}' \
-i http://localhost/rest/node/123456789ABC_0_0_1/data
```
#### Sample JSON Response
```json
{
  "result": 1,
  "data": {
    "id": "Xwi37Osws6zxkCnD"
  }
}
```

## Devices

<a name="activate" />
### 激活设备
**[GET] /node/:nid/activate**
#### 注释
设备激活，成功返回token，token用于设备调用其他接口的凭证
* `nid` 设备的唯一标识

#### Example
```sh
curl -i http://localhost/rest/node/123456789ABC_0_0_1/activate
```
#### Sample JSON Response
```json
{
	"result": 1,
	"data": {
		"token": "6b211615-9761-4a88-94df-4ff3a52eb546"
	}
}
```
-----------------------
<a name="postdata" />
### 发送数据
**[POST] /node/:nid/data**
#### 注释
此接口需带上`node_token`作为请求凭证去请求接口，访问接口时，可以把它们附加到 HTTP 头的 
`X-dolink-Token` 或者以 `access_token=<token>` 的形式附加到 URL 访问地址上。

`node_token`设备激活`[GET] /node/:nid/activate`接口时获得
* `nid` 设备的唯一标识

#### Example
```sh
curl -X 'POST' \
-H 'X-Ninja-Token: YOUR_NODE_TOKEN' \
-H 'Content-Type: application/json' \
-d '{ "G": "0101", "V": 0, "D": 31, "DA": "30", "TIMESTAMP": "1450342488168" }' \
-i http://localhost/rest/node/123456789ABC_0_0_1/data
```
#### Sample JSON Response
```json
{
	"result": 1
}
```
